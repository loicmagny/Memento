Commandes PHP

- Balise PHP: [<]?php.....php[>]

- Créer une boucle: while + endwhile

- Créer conditions: else/if/for/elseif

- Créer un tableau : array

- Afficher variable, entier, décimal: echo

- Afficher les informations structurées d'une variable: var_dump

- Remplacer occurences d'une chaîne: str_replace

- Créer fonction : function

- Renverse une chaîne de caractères: echo strrev

- Trier les tableaux par ordre alphabétique: asort ()
