Commandes HTML

- Faire un tableau: table; thead(th); tfoot(td); tbody(tr);

- Fusionner cellules: colspan (horizontal); rowspan (vertical)

- div id="élément": rendre élément modifiable par CSS (#élément{})

- Faire une liste: numérotée (ol puis il); simple (ul puis il)

- Inclure une image: IMG src= [URL]; +alt: description; +title: titre

- Mettre couleur: span style= color:"couleur"

- Mettre en gras: strong

- Créer un formulaire: label input type="text" name="pseudo" id="pseudo" placeholder="Ex : Zozor" size="30" maxlength="10"

- Mettre en italique: em
