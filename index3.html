Utilisation de Git

- Accéder à la configuration de Git: git config

- Ajouter nom d'utilisateur/adresse mail:- git config --user.name "username" | git config --user.email username@hotmail.com

- Vérifier les paramètres: git --list

- Valider les modifications: git commit

- Ajouter quelque chose: gitt add

- Cloner quelque chose depuis une URL: git clone + url

- Vérifier l'état des fichiers: git status

- Ignorer des fichiers: .gitignore

- Visualiser ce qui a été modifié mais pas encore indexé: git diff

- Visualiser ce qui a été indexé jusq'a maintenant: git dif --cached

- Effacer un/des fichiers: git rm

- Déplacer un/des fichiers: git mv

- Visualiser l'historique des modifications: git log

- Afficher l'historique des modifications avec l'ensemble des modifications effectuées: git log -p --word-diff

- Afficher les statisques de chaque commit: git stat

- Visualiser les modifications via une interface graphique: gitk

- Modifier le dernier commit: git commit --ammend

- Désindexer un fichier déjà indexé: git reset HEAD

- Créer nouvelle branche: git branch

- Switcher de branche: git checkout

- créer dépositoire sur point de dépot: git remote add origin + URL

- Transférer commit dépositoire -> PC: git push origin + NomDuCommit

- Fusionner branches: git merge + NomBrancheAFusionner NomBrancheActuelle

- Transférer commit PC -> Dépositoire: git pull
